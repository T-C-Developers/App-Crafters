# QuickConnect

**Offline Bluetooth Chat & File Sharing App for Android**

QuickConnect is a Kotlin-based Android app that allows users to chat and share files with nearby devices using Bluetooth — no internet required.

## 🔑 Features

- Bluetooth device discovery and secure connection  
- Real-time direct messaging  
- File/image sharing with previews  
- Broadcast messages to multiple devices  
- Local chat/file history with RoomDB  
- Clean, modern UI with customizable settings  

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **Architecture:** MVVM  
- **Bluetooth:** Android Bluetooth API  
- **Database:** Room (SQLite)  
- **UI:** XML (Material Design) 


## 🚀 Getting Started

1. Grant Bluetooth permissions  
2. Connect to a nearby device  
3. Start chatting or sharing files/images  
4. Use the "Blogs" section for broadcast messages  
5. Customize your settings  

## ℹ️ Notes

- Works only when devices are in Bluetooth range  
- Limited multi-device support due to protocol
